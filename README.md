# Pumpkin's Purrfect Meme Picker

A fun cat meme picker that filters images based on your current emotion!

## Features

- Select from 9 different emotions (moody, happy, sad, confused, etc.)
- Filter by animated GIFs only
- Display multiple matching cat memes in a gallery view
- Click outside modal or X button to close

## How to Use

1. Select your current emotion from the radio buttons
2. (Optional) Check "Animated GIFs only" for GIF results
3. Click "Get Image" button
4. View your cat meme(s) in the modal gallery
5. Click outside the modal or the X button to close
